# Mailbox-Setup-Screenshots

Lege hier die Screenshots für die Verbindungs-Anleitung ab (Settings → E-Mail-Konten → Konto verbinden).
Sie erscheinen automatisch im jeweiligen Provider-Guide. Empfohlen: ~16:10, PNG.

- `google.png`    — Google App-Passwörter-Seite (myaccount.google.com/apppasswords)
- `microsoft.png` — Azure App-Registrierung, API-Berechtigung „Mail.Send"
- `smtp.png`      — SMTP-Zugangsdaten beim Anbieter (Host/Port/User/Pass)

Solange eine Datei fehlt, zeigt der Guide einen dezenten „Screenshot folgt"-Platzhalter.
